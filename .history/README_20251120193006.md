# NoSQL Injection Lab
A tiny, intentionally vulnerable Node app for NoSQL injection. 

## Setup

1. Clone the repo:

```bash
git clone https://github.com/yourusername/nosql-injection-lab.git
cd nosql-injection-lab
```

2. Run with Docker Compose:

```bash
docker-compose up --build
```

---

## Services

* `mongo:6.0` – MongoDB, port 27017
* `cassandra:4.1` – Cassandra, port 9042
* `app` – Node.js lab, port 3000

---

Access in browser:

```
http://localhost:3000
```

Login endpoints:


* POST /level1/login — uses {$where: "..."} (string‑interpolated JS).
* POST /level2/login — passes request body directly to User.find().
* POST /level3/login — string‑concatenated CQL query.
